# 포트폴리오 웹사이트

Next.js와 TypeScript를 사용하여 구현한 개인 포트폴리오 웹사이트입니다.

🔗 [포트폴리오 웹사이트 바로가기](https://kwangsoo-seol.web.app/)

## 기술 스택

- **Frontend**: Next.js 14, TypeScript, TailwindCSS, ShadcnUI
- **Animation**: Framer Motion
- **Deployment**: Firebase Hosting

## 주요 기능

- 반응형 디자인
- 다크모드 지원
- 스크롤 프로그레스 바
- 스크롤 스파이 네비게이션
- 프로젝트 카드 애니메이션
- 섹션별 페이드인 효과
